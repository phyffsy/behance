# Simple Bakery App

Projeto Expo + React Native + TypeScript baseado nas telas do app mobile de panadería.

## Rodar

```bash
npm install
npx expo start
```

## Estrutura

- `src/screens`: telas em `.tsx`
- `src/components`: componentes reutilizáveis
- `src/constants`: cores do template
- `src/data`: produtos e textos
- `assets`: imagens usadas no app
